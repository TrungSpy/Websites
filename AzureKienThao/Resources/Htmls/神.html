神

common

シン/SHIN/    ジン/JIN/    かみ/kami/    かん-/kan-/    こう-/kō-/SHIN/シン/    JIN/ジン/    kami/かみ/    kan-/かん-/    kou-/こう-/

gods;  mind;  soul divinité;  esprit;  âme Deus;  mente;  alma;  divindade;  divino dios;  alma

Radical: 礻 (spirit).   Strokes: 9画.   Elements: ｜日礻田.   Pinyin: shén.   Hangul: 신 [sin].   Nanori: か / かぐ / かな / かも / くま / こ / こは / だま / みka / kagu / kana / kamo / kuma / ko / koha / dama / mi.

Jōyō Kanji 3rd Grade.   JLPT N3.

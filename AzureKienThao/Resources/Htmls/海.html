common

カイ/KAI/    うみ/umi/KAI/カイ/    umi/うみ/

sea;  ocean mer;  océan oceano;  mar mar;  océano;  playa

Radical: 水 (water).   Strokes: 9画.   Elements: 氵母毋.   Pinyin: hǎi.   Hangul: 해 [hae].   Nanori: あ / あま / うな / うん / え / か / た / ひろ / ひろし / ぶ / まち / まま / み / め / わたるa / ama / una / un / e / ka / ta / hiro / hiroshi / bu / machi / mama / mi / me / wataru.

Jōyō Kanji 2nd Grade.   JLPT N4.

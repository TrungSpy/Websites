フク/FUKU/FUKU/フク/

blessing;  fortune;  luck;  wealth bénédiction;  fortune;  chance;  richesse benção;  fortuna;  sorte;  riqueza fortuna;  suerte;  dicha

Radical: 礻 (spirit).   Strokes: 13画.   Elements: 一口田礻.   Pinyin: fú.   Hangul: 복 [bog].   Nanori: とし / とみ / ふ / ふき / ふっ / ぼく / よしtoshi / tomi / fu / fuki / fu' / boku / yoshi.

Jōyō Kanji 3rd Grade.   JLPT N3.

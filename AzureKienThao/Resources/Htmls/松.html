common

ショウ/SHŌ/    まつ/matsu/SHOU/ショウ/    matsu/まつ/

pine tree pin pinheiro pino

Radical: 木 (tree).   Strokes: 8画.   Elements: ハ木厶.   Variants: 枩柗.   Pinyin: sōng.   Hangul: 송 [song].   Nanori: おお / しょ / ま / まっō / sho / ma / ma'.

Jōyō Kanji 4th Grade.   JLPT N1.

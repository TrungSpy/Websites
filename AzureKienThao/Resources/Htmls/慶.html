ケイ/KEI/    よろこ.び/yoroko.bi/KEI/ケイ/    yoroko.bi/よろこ.び/

jubilation;  congratulate;  rejoice;  be happy jubiler;  se réjouir;  féliciter;  être heureux júbilo;  congratular;  regozijar-se;  estar feliz júbilo;  felicidad;  diversión;  regocijo

Radical: 心 (heart).   Strokes: 15画.   Elements: 心冖夂广.   Pinyin: qìng.   Hangul: 경 [gyeong] 강 [gang].   Nanori: き / きよん / け / みち / む / やす / よしki / kiyon / ke / michi / mu / yasu / yoshi.

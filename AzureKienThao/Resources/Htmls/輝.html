キ/KI/    かがや.く/kagaya.ku/KI/キ/    kagaya.ku/かがや.く/

radiance;  shine;  sparkle;  gleam;  twinkle étincelant;  étincelle;  briller;  scintiller brilho;  luzir;  centelha;  faiscar;  piscar resplandor;  destello;  centelleo;  resplandecer;  destellar;  centellear

Radical: 車 (car).   Strokes: 15画.   Elements: 車⺌儿冖.   Pinyin: huī.   Hangul: 휘 [hwi].   Nanori: あき / あきら / さき / てる / ひ / ひかるaki / akira / saki / teru / hi / hikaru.

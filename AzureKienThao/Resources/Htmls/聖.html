聖

common

セイ/SEI/    ショウ/SHŌ/    ひじり/hijiri/SEI/セイ/    SHOU/ショウ/    hijiri/ひじり/

holy;  saint;  sage;  master;  priest saint;  sage;  maître;  prêtre santo;  sagrado;  sábio;  mestre;  padre santo;  sagrado;  sabio;  relacionado con cristianismo

Radical: 耳 (ear).   Strokes: 13画.   Elements: 王口耳.   Variants: 垩.   Pinyin: shèng.   Hangul: 성 [seong].   Nanori: きよ / さと / さとし / せ / たか / ただ / ひろ / まさ / みkiyo / sato / satoshi / se / taka / tada / hiro / masa / mi.

Jōyō Kanji 6th Grade.   JLPT N1.
